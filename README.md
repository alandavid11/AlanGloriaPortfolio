# Alan Gloria Portfolio

Personal portfolio for Alan Gloria, an iOS Engineer focused on native Apple platforms, production delivery, automated quality, and AI-assisted engineering workflows.

## Tech Stack

- React 19
- TypeScript
- Vite
- Framer Motion
- Tailwind CSS
- Three.js / React Three Fiber for lightweight visual effects
- Vercel deployment

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```
